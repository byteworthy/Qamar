# Android Google Play Screenshot Specifications

## Required Sizes
- Phone: Min 320px, max 3840px (recommend 1440 x 3120 px for Pixel 8 Pro)
- 7-inch tablet: Min 1024 x 500 px (recommend 1600 x 2560 px)
- 10-inch tablet: optional

## Screenshot Requirements
- 2-8 screenshots required for phone
- Same content as iOS but with Android device frames

## How to Take Screenshots
1. Run: `npm run android` (opens in emulator)
2. Navigate to target screen
3. Screenshot: Ctrl+S or use extended controls in emulator
4. Apply Pixel 8 Pro device frame in Figma

## Feature Graphic (Required by Google Play)
- Size: 1024 x 500 px
- Content: App name + tagline + 3-4 key feature highlights
- Background: Noor cream (#FAF8F3) or dark background
- Include: Noor logo, "Your Complete Islamic Companion" tagline
- Create in Figma or Canva
