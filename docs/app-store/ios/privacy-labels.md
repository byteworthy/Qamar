# iOS Privacy Nutrition Labels

## Data Not Collected
Qamar does not collect any data from users.

Answer all questions as "No" / "We do not collect this data":
- Contact Info: No
- Health & Fitness: No
- Financial Info: No
- Location: No (location is processed on-device only, never sent to servers)
- Sensitive Info: No
- Contacts: No
- User Content: No (recordings processed on-device)
- Browsing History: No
- Search History: No
- Identifiers: No
- Usage Data: No
- Diagnostics: No

## App Privacy Details
- Location: Used only for prayer time calculations. Processed on-device. Never sent to our servers.
- Microphone: Used only for pronunciation recording. Audio is NOT stored or transmitted.
- All user preferences, progress, and data: Stored locally on device only.
- No analytics SDKs (no Firebase Analytics, Amplitude, Mixpanel, etc.)
- Sentry error tracking: Only captures crash data, no PII
