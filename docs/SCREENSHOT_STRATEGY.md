## Text Overlay Design Guidelines

### Typography
- **Primary Font:** SF Pro Display (iOS native) or Roboto (Android)
- **Header Text:** 40-50px, Bold weight, White color
- **Subtitle Text:** 24-28px, Regular weight, White or Light Gray
- **Alignment:** Centered for all overlays
- **Shadow:** Subtle black shadow (0px 2px 8px rgba(0,0,0,0.3)) for readability

### Color Palette (Overlay Backgrounds)
- **Gradient Option 1:** Warm Islamic (Gold #D4AF37 â†’ Warm Beige #F5E6D3)
- **Gradient Option 2:** Trust (Deep Purple #6B4FA0 â†’ Soft Blue #5A9FD4)
- **Gradient Option 3:** Calm (Sage Green #8FA998 â†’ Soft Mint #C1E5D6)
- **Solid Option:** Semi-transparent dark overlay (rgba(0,0,0,0.5)) with white text

### Placement
- **Top Third:** Main value proposition (larger text)
- **Bottom Third:** Supporting detail or feature (smaller text)
- **Never:** Cover critical UI elements (buttons, key visuals)

### Branding
- Include small Noor logo in corner (optional)
- Use Noor brand colors consistently (warm neutrals with gold accents)
- Maintain Islamic aesthetic (geometric patterns, calligraphy touches)
