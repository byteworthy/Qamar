## Review Questions Checklist

Track which validation questions have been addressed:

### Theological Correctness
- [ ] Quranic translations verified (Sahih International acceptable?)
- [ ] Hadith citations confirmed authentic
- [ ] Arabic text accuracy checked

### Therapeutic Application
- [ ] Appropriate use of ayat/hadith in mental health context confirmed
- [ ] Therapeutic explanations deemed respectful
- [ ] Staying within proper Islamic boundaries confirmed

### Representation
- [ ] Accurate representation of Islamic teachings verified:
  - [ ] Emotional wellness (sabr, tawakkul, shukr)
  - [ ] Self-compassion approaches
  - [ ] Dealing with anxiety, grief, shame
  - [ ] Hope and mercy in difficulty

### Sensitive Areas
- [ ] Crisis handling approach approved (no verses in crisis)
- [ ] Scrupulosity (waswas) handling reviewed
- [ ] Mental health boundaries appropriate
