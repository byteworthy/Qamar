3. Optimize file sizes (< 1MB each)

### Step 3: Add Schema Markup (15 minutes per page)

For each page:
1. Copy schema templates from guide
2. Customize FAQ questions per page
3. Add to bottom of `<body>` section
4. Validate with Google Rich Results Test

- [ ] Rich Results Test (Google)
- [ ] Open Graph Debugger (Facebook)
- [ ] Twitter Card Validator
- [ ] Mobile-Friendly Test

### Step 5: Submit to Google (10 minutes)

2. Submit sitemap to Google Search Console
3. Request indexing for new pages
4. Set up position tracking for target keywords
