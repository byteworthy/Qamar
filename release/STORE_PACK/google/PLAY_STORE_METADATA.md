## Data Safety Section

### Data Collected

**Personal Information**
- Email address (optional, for account/support purposes only)

**App Activity**
- User-generated content (journal entries, tutor conversations, Ask Karim questions)
- Audio recordings (pronunciation coach — processed for feedback)
- App interactions (feature usage for diagnostics)

**App Info and Performance**
- Crash logs and diagnostics
- Device type and OS version

### Data Not Collected
- Precise location
- Contacts or address book
- Phone number
- Social media data
- Photos or videos (beyond pronunciation audio)
- Device advertising ID
- Financial or payment info (handled entirely by Google Play)
- Browsing or search history outside the app

### Data Use Purposes
- **App Functionality**: User content (text, audio) processed to generate tutor responses, pronunciation feedback, Islamic knowledge answers, reflections, and translations
- **Analytics**: Basic device diagnostics for app stability
- **Account Management**: Optional email for support requests
- **No Advertising**: No data used for advertising
- **No Third-Party Marketing**: No data shared with third parties for marketing

### Third-Party Data Processing
- **Anthropic Claude API**: User-generated text content and audio transcriptions are sent to Anthropic's API for processing (tutor conversations, reflections, pronunciation feedback, translations, Ask Karim responses). Anthropic does not use API data for model training per their data usage policy. Data is processed per-request and not retained by Anthropic beyond the API call.

### Data Sharing
- No third-party advertising or behavioral sharing
- No sale of personal data
- Third-party processing limited to Anthropic API for core app functionality (see above)

### Security Practices
- Encrypted transmission (HTTPS/TLS)
- Encrypted storage at rest
- Users can request data deletion

### Data Retention & Deletion
- User content (entries, conversations) retained until user deletes
- Audio recordings processed for feedback and not retained long-term
- Session logs retained per retention policy (30 days)
- Subscription status managed by Google Play

### Data Deletion Requests
- Users can delete their content from within the app
- Full data deletion available on request via support@byteworthy.com
- Subscription cancellation handled in Play Store
