## Reviewer Summary (Paste in "App Review Information" → "Notes" field)

Noor is a self-reflection and journaling app for adults (18+).

### Core Functionality
Users write journal entries and receive AI-generated reflective responses designed to support personal self-reflection. The app uses cognitive behavioral techniques to help users organize their thoughts.

### AI Usage Disclosure
The app uses OpenAI's API to process user input and generate personalized reflective responses. This AI processing is:
- Disclosed in the app description
- Disclosed during onboarding (Privacy & Safety screens)
- Essential to app functionality
- Limited to reflection and journaling support

### What This App Is NOT
- NOT therapy or counseling
- NOT medical advice or diagnosis
- NOT a crisis intervention tool
- NOT involving licensed clinicians
- NOT making health claims
- NOT collecting protected health information (PHI)

### Safety Disclaimers
The app includes prominent safety disclaimers:
- Onboarding screen explicitly states "not therapy" and "not for emergencies"
- Crisis resources displayed (988 Suicide & Crisis Lifeline, Crisis Text Line, 911)
- App description clearly states limitations
- Privacy Policy discloses AI processing

### Privacy & Data Handling
- User journal entries processed by OpenAI API for reflection generation
- No data selling or advertising
- No location tracking
- No health data collection
- Data deletion available on request
- Privacy Policy URL: https://byteworthy.github.io/noor-legal/privacy-policy

### Age Rating Justification
Rated 18+ (Adults Only) due to:
- Mental health reflection content
- Responsibility required for self-directed journaling
- Crisis disclaimer appropriate for mature users
